Flexget教程——全自动下载新番指南

虽然正版新番被查水表，但是没有人会担心没有新番可看，因为毕竟国内动漫的真正基石是各大字幕组和汉化组，现在正是回归bt下载的好时机。bt下载一点也不麻烦，用Flexget可以做到全自动更新+下载+邮件通知，只需要每个季度更新下新番列表就行了。
整个指南只需要四步：

  1. 下载安装Flexget
  2. 安装transmission或者迅雷离线脚本
  3. 将想追的新番写入配置文件
  4. 运行

下载安装Flexget

Flexget是一个自动寻找、抓取、分析下载内容的工具，可以自动输出给几十种下载工具来下载。官网：http://flexget.com/

你也可以在Windows或者OSX上使用，不过也需要先安装python-pip,可以点击这里观看官方指南。

当然在PC上安装了Flexget也不可能用来自动下载新番的，因为不能全天候开机。所以你需要一台全天候开机的设备，比如路由器，树莓派，手机。

无论是那种设备，最重要的是你要能够安装上python和python-pip，之后就可以通过pip install flexget来安装flexget。

路由器

重点讲一下如何在路由器上部署Flexget，毕竟路由器才是最普及的设备。你的路由器要能够挂载U盘或者移动硬盘，否则白搭。

  • 首先要给你的路由器刷上openwrt系统，由于路由器种类繁多，请参考openwrt支持设备列表来看看自己的路由器是否适合刷机。本人推荐各位刷Pandorabox，一个国人的openwrt的发行版本，已经内置了绝大多数国人需要的功能，到这里来查看系统支持设备列表。

  • 如果你刷入openwrt后，剩余足够多的空间（现在的国产智能路由器都能做到），而且软件源是可用的，那么按照官方教程操作即可。

  • 如果你的设备过老，空间不足，那么可以把软件安装到U盘或者移动硬盘。以安装python为例：

安装transmission或者迅雷离线脚本

首先你需要了解，BT下载需要有公网IP地址才好使。有公网IP地址的用户被称为highID，没有公网IP地址的用户被称为lowID，两个lowID之间无法直接通讯，lowID只能从highID那里下载。而且很无解的一点是，上级网关设备完全可以禁止你的P2P，只需要针对相应的下载软件禁用uPnP的相应端口映射即可（这个常见于校园网/公司内网）。
所以如果你的网络环境是电信/联通而且有公网IP地址，那么恭喜你了，你可以使用在路由器上通过transmission下载，全自动且方便快捷。如果你是没有公网IP地址的家庭宽带、校园网或者公司内网，推荐使用迅雷离线脚本，当然，缺点是要交会员费，另外无法解决偶尔出现的要求输入验证码的情况。

Flexget的配置文件为～/.flexget/config.yml
